1         Overview

This document addresses the steps necessary to setup the TeamPoS3000 XT Kiosk system devices using a fresh installation of Microsoft Windows XP SP2.

 

A folder will be present on all FTXS created operating system images named:

C:\TeamPoS3000-XT_Software_Support

 

This folder contains documentation and driver files for installation and use. This folder would be current to the image creation date. The Fujitsu Transaction Solutions support website should be checked for updates.

 

http://www.fujitsu.com/us/services/retailing/support/drivers/teampos3000xl/

 

2         TeamPoS3000 XT Hard disk Subsystem

 

The TeamPoS® 3000 hard disk subsystem utilizes Serial ATA (SATA) hard drives controlled by an on-motherboard Promise 579 SATA RAID controller. While the motherboard supports RAID the TeamPoS3000 XT only supports one disk drive making RAID of little value. The RAID function is used in the  TeamPoS® 3000 XL.

 

3         Initial TeamPoS3000 XT Setup

 

Installing Windows XP SP2 from release media. As the figure below shows, the hard disk interface is via the on-motherboard Promise 579 Serial ATA (SATA) RAID controller. This requires use of the F6 method to install additional critical storage drivers (the Promise drivers) during the Windows setup process. This method is documented in Windows setup help and readme files that accompany Microsoft Windows XP retail media.  TeamPoS3000 XT Windows XP images received from FTXS Foothill Ranch will already have gone through this process. A USB Floppy drive must be used to load the SATA driver via the F6 process when loading Windows XP from Microsoft Release Media. A USB HUB must also be used as three USB devices are needed (Keyboard, CD/DVD reader, and Floppy). A USB HUB is needed as only USB ports C and D are supported by the BIOS. The remaining USB ports are not active until Windows is operational.

 

It should be noted that Promise Drivers version 1.00.0.36 should be utilized for the initial Windows XP Pro installation. These drivers are Windows Logo signed. Use of, newer, unsigned versions of the Promise driver have proven problematic during installation, though functioning correctly in operation. The recommended steps would be to install version 1.00.0.36 drivers during operating system installation, followed by upgrading to latest drivers once installation is complete.

 

 

Also note that when using the 133UQ keyboard and USB floppy together, the TeamPoS3000 XT Kiosk system may have to be unplugged and then plugged back into the AC outlet for the 133UQ keyboard to be initialized in time for the F2 or F12 keys to be used to enter the BIOS.

 

The following instructions assume Windows XP Pro SP2 has been successfully loaded.

 

 

Once Windows XP Pro is installed and loads to the desktop, a look at the Device Manager will show several devices not yet setup, as depicted above.

 

The remainder of this document describes how to load the drivers needed to support the custom devices embedded on the motherboard of the TeamPoS3000 XT Kiosk system. In addition to these drivers, a touch screen driver must be loaded to support the touch screen. While the touch screen will partially function after loading the Microsoft Windows XP release media, the touch screen will not perform correctly until the touch screen driver is installed and the touch screen calibrated. Refer to the document found in folder C:\TeamPoS3000-XT\Drivers\LCD Display Touch\TeamPoS 3000 D25\Documentation for the TeamPoS3000 XT Kiosk system with 15 display and C:\TeamPoS3000-XT\Drivers\LCD Display Touch\TeamPoS 3000 D22\Documentation for the TeamPoS3000 XT Kiosk system with a 12 display.

4          Video

 

The TeamPoS3000 XT utilizes the on-motherboard Intel 855GME chipset, which support dual-head video. An analog CRT video connector is located on the controller backplane, in addition to the internal flat panel display connector.

 

By adding an analog CRT to the backplane, the Kiosk LCD (Notebook) and analog monitor can be set in the Clone or Extended mode. In the Clone mode the same data is displayed on both displays. In the Extended mode, one display becomes an extension to the right of primary display. The mouse pointer can be moved from the primary display to secondary display by moving the cursor to the right.

4.1           Video Driver Installation

 

 

  1. Locate the folder containing the TeamPoS3000 XT Video Drivers. In this example, the drivers are located in c:\TeamPoS3000-XT_Software_Support\Drivers\Mother Board\Extreme Graphics\Driver. Standard Intel Extreme Graphics drivers are used.

 

Run win2k_xp1418.exe from this folder.

 

  1. Screen will blink several times. This single installation instance installs support for both video channels. Restart the computer at the end of the process.

 

 

 

 

  1. When the Intel® Chipset Graphics Driver Software screen appears, click Next.

 

 

Figure 1

 

  1. At the Welcome screen, click Next.

 

 

 

  1. The License Agreement will then be displayed.  Click Yes to continue the driver loading.

 

 

  1. When driver installation has completed, select Yes, I want to restart my computer now and Finish to exit.

 

  1. Note:  Once the restart is performed the Graphics Driver will only have set up one display. Also, the resolution may need to be configured to optimal settings for the display type. If dual displays are used, follow the procedure listed in Configuring Multiple Displays.

 

 

 

  1. After the system reboots the following information will be displayed in the Device Manger for the Display adapter. This completes the installation of the video drivers.

 

4.2           Video Display Configuration

 

Use Intel Extreme Graphics video setup utility to manage video settings. The utility can be launched from the task tray icon Intel Extreme Graphics 2 for Mobile; select "Graphics properties", or launch from Display Control Panel/Settings/Advanced/Intel Extreme Graphics 2 for Mobile/Graphics Properties

 

 

Graphics modes, such as single, extended, etc., can be manipulated from this screen. Display properties, such as resolution, can be found by clicking the Display Settings tab on the left navigation pane.

 

Note the entries under the Single Display area in the picture above. Up to two displays are itemized, Monitor and Notebook. The above screen shows what one would see when the TeamPoS3000 XT is configured with a Analog Display (Monitor) connected in addition to the Kiosk LCD (Notebook).

 

·        Monitor refers to the display connected via the analog (VGA) connection on the bottom of the TeamPoS3000 XT.

 

·        Notebook refers to the LVDS (Low Voltage Differential Signaling) display connection that is used with the TeamPoS3000 XT KIOSK units. The Notebook display type is itemized even though no display device is attached because the internal circuitry to drive this display is present in the Intel Chipset. This is the method typically used in notebook/laptop configurations, and used in TeamPoS3000 XT KIOSK.

 

When the desired video mode is setup, proceed to install and run the Display_Select set video mode utility as described in the following section.

 

4.3           Display_Select Utility

 

This utility only needs to be loaded when operating the TeamPoS3000 XT in dual monitor mode. This utility is needed to provide error recovery in the event the second Monitor is unplugged during a reboot. The program Install_KIOS.bat will place the program Display_Select in the program folder. This program allows the end user to set the error recovery mode. The program SetDisp.exe is setup to run on boot to place the graphics adapter in the desired mode.

 

 

1.      To install the Display_Select utility, navigate to the folder location containing the Set Video Mode Install_KIOSK.bat file. In this example, the folder is c:\TeamPoS3000-XT_Software_Support\Drivers\Mother Board\Extreme Graphics\Set Video Mode. Select and run the file Install_KIOSK.bat.

 

 

2.      The Display_Select utility program has been added to the Start Menu, and the SetDisp.exe program added to the system to run automatically at system start-up. Select and start the Display_Select utility from the Start Menu.

 

 

3.      The Display_Select utility presents a single, simple screen. Select the desired video mode from the list provided, and click "Register". The video mode is now saved and will be used by SetDisp.exe as the benchmark comparison at system start-up.

2.     Select No, not this time and then Next.

 

 

3.     Check Install from a list or and click Next

 

 

4.     Check Include this location in the search: and browse to the folder that contains the Intel LAN driver. Using the FTXS folder layout the LAN driver is located in C\TeamPoS3000-XT_Software_Support\Drivers\Mother Board\LAN\Driver. Uncheck Search removable media (floppy, CD-ROM) and then select Next

 

 

5.     Selecting Finish will finalize the Intel LAN driver installation.

 

 

 

6.     The Device Manager will now display the above information for the Network Adapter.

7         PCI Bridge Device and PCI Serial Ports

With the TeamPoS3000 XT configured with the default BIOS settings, the OXFord PCI to COM port chip is enabled. If the drivers for this chip are loaded then COM3-5 will appear in the Device Manager.  Since there are no physical connectors for COM3-5 on the TeamPoS3000 XT, these COM ports can not be used. These COM ports can only be used when the motherboard is in the TeamPoS3000 XL. The figure below shows the Device Manager before the COM3-5 drivers are installed:

 

 

At this point, all we see are COM1-2.

 

Since the COM3-5 logic is not used on the TeamPoS3000 XT the system integrator has the following three choices when loading Windows XP onto the TeamPoS3000 XT.

 

-          Do nothing  Leave the Other PCI Bridge Device and PCI Serial Port as undefined and in the folder Other Devices.

-          Disable the OXFord PCI Bridge in the BIOS which will prevent Windows XP from detecting the hardware. See section 7.1

-          Load the PCI Bridge and PCI Serial Port driver to set up COM3-5 which can not be accessed. See section 7.2

 

7.1           Disabling PCI Bridge to COM3-5

 

You must enter the BIOS and disable the OXFord COM Controller. Follow the following steps:

 

1.      Restart the TeamPoS3000 XT and enter BIOS setup by pressing F2 during BIOS POST.

2.      Enter the Integrated Peripherals screen.

3.      Enter Onboard Device screen

4.      In the Onboard Device screen, disabling the OXFord COM Controller will disable the PCI Bridge (COM3-5).

5.      Save and exit the BIOS setup utility

 

7.2           Loading PCI Bridge and COM3-5 drivers

 

 

1.     In Device Manager/Other Devices, right-click Other PCI Bridge Device and select Update Driver.

 

 

2.     Select No, not this time and then Next.

 

 

3.     Select Install from a list or specific location (Advanced) and then Next.

 

 

4.     At the Choose search and installation options screen, check "Include this location in the search" and browse to the folder containing the port drivers, in this example, c:\TeamPoS3000-XT_Software_Support\Drivers\Mother Board\PCI-COM3-5\Driver, and then click Next.

 

 

5.     Select Continue Anyway.

 

 

6.     Selecting Finish will complete the installation of the PCI bridge.

 

 

7.     In Device Manager/Other Devices, right-click Other PCI Serial Port and select Update Driver.

 

 

 

8.     Select No, not this time and then Next.

 

 

 

9.     Select Install from a list or specific location (Advanced) and then Next.

 

 

10. At the Choose search and installation options screen, check "Include this location in the search" and browse to the folder containing the port drivers, in this example, c:\TeamPoS3000-XT_Software_Support\Drivers\Mother Board\PCI-COM3-5\Driver, and then click Next.

 

 

 

11. Select Continue Anyway.

 

 

12. Select No, not this time and then Next.

 

 

 

13. Select Install from a list or specific location (Advanced) and then Next.

 

 

14. At the Choose search and installation options screen, check "Include this location in the search" and browse to the folder containing the port drivers, in this example, c:\TeamPoS3000-XT_Software_Support\Drivers\Mother Board\PCI-COM3-5\Driver, and then click Next.

 

 

 

15. Selecting Finish will complete the PCI Bridge installation.

 

 

16. Selecting Finish will complete the PCI Serial installation.

 

 

 

17. At this point, Device Manager shows device OX16PCI95x PCI bridge installed under the Multifunction Adapters branch and under Ports as COM3-5.

 

 

 

8         Backplane Connectors

 

The picture below shows the TeamPoS3000 XT motherboard backplane. Unlike the TeamPoS3000 XL there are no optional add-on I/O cards.

 

 

The following describes the connectors from left to right and top to bottom.

-          The VGA connector provides for the connection of a second analog display. This display can be used in either the Extended mode with the Kiosk LCD or Clone Mode with the KIOSK LCD.

-          Non powered COM ports COM1 (1) and COM2 (2).

-          USB A, B, C, D - Four traditional non-powered USB ports, with only ports C and D being operational during the boot process. USB ports C and D operate as USB 1.1 during boot and all four ports operate as USB 2.0 during Windows.

-           Powered USB ports E and F providing 5 volts, operating as USB 2.0 ports.

-           Powered USB port G providing 12 volts, operating as a USB 2.0 port.

-          The LAN connector provides 10/100 Ethernet support.

-          The DUMP switch is currently not used.

-          The RESET button will reboot the TeamPoS3000 XT.

-          The PORT ON/OFF switch is currently not used.
